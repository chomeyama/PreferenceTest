Experimental Procedure

  1. Enter your name and area you live.
  2. Press "Start Experiment" button.
  3. Listen to the displayed audios and select one which you think has better sound quality.
  4. Press "next" button and start next evaluation.
  5. When you finish the last evaluation, press "finish" button and download the result.

  6. Email : authors_email@address

Notes

  • This experiment takes about 10 minutes.
  • Please do not update the page during this experiment.
  • If you are using a smartphone, you may not be able to download the result, so we recommend using a PC.
  • Please conduct this experiment using headphones or earphones in a silent place.
  • All information obtained in this experiment will not be used for any purpose other than this experiment.
